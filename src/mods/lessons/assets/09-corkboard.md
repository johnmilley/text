# corkboard — a folder as index cards

Right-click a folder (or the sidebar background) → **corkboard**, or press
`Ctrl+Shift+D` for the folder of the note you're reading. Every note
becomes an index card: its title over a one-line synopsis. Try it on this
`lessons` folder now, then come back.

## synopses

A card shows the note's `synopsis:` frontmatter when there is one — this
note's card reads "a folder as index cards…" because of the line at the top
of this file — and falls back to the note's first words otherwise. Click
the pencil on a card to write a synopsis without opening the note; it's
saved into that note's frontmatter, so it travels with the file.

## rearranging

Drag cards to reorder them. The order lives in a hidden `.corkboard` file
in the folder — never in your notes — and the **sidebar follows it**:
shuffle chapters on the board and the file tree shows the same sequence.
Files the board doesn't show (images, PDFs) keep sorting after the ordered
ones.

Folder cards drill down into their own board, with a breadcrumb trail back
out. The `+` card starts a new note in the folder you're looking at.

## what it's for

Seeing structure at arm's length. Planning a book: one card per chapter,
shuffle until it reads right (`03-illustrated-book/` is a ready-made folder
to practice on). Outlining an essay: one card per argument. Running a
course: one card per week. The synopsis line enforces a useful discipline —
if you can't say what a note is *for* in one line, the note doesn't know
either.
